# deltarun apps

A small collection of **Progressive Web Apps (PWAs)** — web apps you can install on your phone and use like a normal app, even offline.

**Open the apps:** https://deltarun28.github.io

## Apps

| App | Link |
|-----|------|
| Study Quiz — practice quizzes across trades and topics | https://deltarun28.github.io/quiz/home.html |
| Iron Vale — real-time territory strategy game | https://deltarun28.github.io/iron-vale/ |

## Install on your phone

No app store needed — just open the link and add it to your home screen:

- **iPhone / iPad (Safari):** tap the **Share** button → **Add to Home Screen**.
- **Android (Chrome):** tap the **⋮** menu → **Install app** (or **Add to Home Screen**).

It then opens fullscreen with its own icon and works offline once loaded.
